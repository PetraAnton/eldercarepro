# Bảng đánh giá hoạt động sinh hoạt hằng ngày mở rộng (IADL – Lawton Instrumental Activities of Daily Living)

| Hạng mục | Điểm | Tiêu chí |
|---|---:|---|
| **Sử dụng điện thoại** | 0 | Không sử dụng được |
|  | 1 | Gọi / nhận cuộc gọi với trợ giúp |
|  | 2 | Tự gọi và nhận điện thoại |
| **Đi mua sắm** | 0 | Không thể đi mua sắm |
|  | 1 | Mua sắm được với trợ giúp |
|  | 2 | Tự lập đi mua sắm |
| **Chuẩn bị bữa ăn** | 0 | Không thể chuẩn bị |
|  | 1 | Chuẩn bị được một phần |
|  | 2 | Tự chuẩn bị hoàn toàn |
| **Làm việc nhà** | 0 | Không làm được |
|  | 1 | Làm việc nhẹ |
|  | 2 | Làm việc nhà đầy đủ |
| **Giặt giũ** | 0 | Phụ thuộc hoàn toàn |
|  | 1 | Tự giặt một phần |
|  | 2 | Tự giặt hoàn toàn |
| **Phương tiện đi lại** | 0 | Không thể di chuyển / phụ thuộc |
|  | 1 | Di chuyển được với trợ giúp |
|  | 2 | Tự lập sử dụng phương tiện công cộng / cá nhân |
| **Quản lý thuốc** | 0 | Không tự quản lý được |
|  | 1 | Cần nhắc nhở / trợ giúp |
|  | 2 | Tự quản lý chính xác |
| **Quản lý tài chính** | 0 | Không quản lý được |
|  | 1 | Quản lý được một phần |
|  | 2 | Tự quản lý hoàn toàn |

---

## Ghi chú
- Tổng điểm tối đa: **16 điểm**  
- Điểm càng cao → mức độ **độc lập trong hoạt động sinh hoạt phức tạp** càng tốt  
- IADL thường dùng để đánh giá **người cao tuổi**, **suy giảm nhận thức**, hoặc **giai đoạn sớm mất chức năng**
