# Bảng đánh giá hoạt động sinh hoạt hằng ngày (ADL – Barthel Index)

| Hạng mục | Điểm | Tiêu chí |
|---|---:|---|
| **Ăn uống** | 0 | Không tự ăn được |
|  | 5 | Cần trợ giúp (cắt, chuẩn bị, chế độ ăn…) |
|  | 10 | Tự lập |
| **Tắm** | 0 | Phụ thuộc |
|  | 5 | Tự tắm được |
| **Vệ sinh cá nhân** | 0 | Cần trợ giúp |
|  | 5 | Tự lập (rửa mặt, đánh răng, chải tóc…) |
| **Mặc quần áo** | 0 | Phụ thuộc |
|  | 5 | Cần trợ giúp |
|  | 10 | Tự lập |
| **Đại tiện** | 0 | Không kiểm soát / phải dùng thụt, chăm sóc hoàn toàn |
|  | 5 | Thỉnh thoảng không kiểm soát |
|  | 10 | Kiểm soát tốt |
| **Tiểu tiện** | 0 | Không kiểm soát / sonde tiểu, phụ thuộc |
|  | 5 | Thỉnh thoảng không kiểm soát |
|  | 10 | Kiểm soát tốt |
| **Đi vệ sinh** | 0 | Phụ thuộc |
|  | 5 | Cần trợ giúp |
|  | 10 | Tự lập |
| **Transfer (Giường – Ghế)** | 0 | Không thể / cần 2 người hỗ trợ |
|  | 5 | Cần nhiều trợ giúp |
|  | 10 | Cần ít trợ giúp |
|  | 15 | Tự lập |
| **Đi lại** | 0 | Không đi lại được |
|  | 5 | Xe lăn phụ thuộc / cần trợ giúp |
|  | 10 | Đi lại có trợ giúp (gậy, khung…) |
|  | 15 | Tự lập |
| **Cầu thang** | 0 | Không thể |
|  | 5 | Cần trợ giúp |
|  | 10 | Tự lập |

---

**Ghi chú:**  
- Tổng điểm tối đa: **100 điểm**  
- Điểm càng cao → mức độ **độc lập chức năng** càng tốt
